# Capstone : 심리전 중심의 방향성 기반근접 전투 게임 개발
## The Knight (더 나이트)
### Development of psychological warfare-oriented direction-based close combat game

해당 작품은 2026 국립창원대학교 컴퓨터공학과 캡스톤디자인(졸업작품)에 제출하기 위해 제작한 게임 작품으로   
적 AI와 1대1 진검 승부를 하는 포 아너 식 전투 게임이자   
심리전 중심의 방향성 기반 근접 전투 게임 입니다.

제작 기간 : 약 2~3개월   
※ 무료 에셋의 저작권 표시는 게임 내부(크레딧)에 표시되어 있습니다.

| 맡은 업무 | 성명 |
| --- | --- |
| 개발 | 이광수 |
| 개발 | 정철진 |  

| 사용한 툴 | 용도 |
| --- | --- |
| Unity 6 | 게임 엔진 및 통합 개발 환경 |
| TRELLIS.2 | 로컬 3D 에셋 생성 AI | 
| Cartwheel & Quick Magic | 모션 캡처 기반 리깅 애니메이션 생성 AI |

| Modddif | 디지털 텍스처링 및 머티리얼 보완 (AI 기반) |
| Mixamo | 범용 애니메이션 플랫폼 활용 |
